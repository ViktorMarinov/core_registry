Import and Export Content

Import and Export the public artifacts from the Registry. Import Content service provide the end-point for importing public registry content from another instance. Export Content service helps in exporting the content of the public registry as a zip file ready for further import.

Clone Instances

Import and Export the whole Repository content including the Users Workspaces and Configurations. Clone Import service provide the end-point for importing cloned content from another instance. Clone Export service helps in exporting the content of the whole repository as a zip file ready for further import.

Import Projects

Import Project service provide the end-point for importing project content in design time format (source). This is useful for constructing a PROD instance (consisting only of Runtime components) by importing one or many ready to use source projects.
